🐾 Animal Exhibitions

This is a React-based web application that showcases different animal exhibitions. The project uses Supabase as the backend for storing and managing data.

🚀 Features

- 🦁 Display list of animal exhibitions
- 📄 View detailed information of each animal
- ➕ Add new animal entries (if enabled in app)
- 🔄 Real-time data handling using Supabase
- 📱 Fully responsive UI design

🛠️ Tech Stack

- React.js
- Supabase (Database + Backend)
- JavaScript
- CSS
